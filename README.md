# Big Five Personality Chat Simulator (性格対話シミュレーター)

性格の5大因子（ビッグファイブ）、性別、年齢、および会話をする環境（バックグラウンド）を設定し、そのキャラクターになりきったAIとチャットができるWebアプリケーションです。

---

## 🌟 コンセプト

> **「アシスタントとして、完璧に『役に立たない』。シミュレーターとして、完璧に『リアル』。」**

このアプリケーションは、一般的なChatGPTやGeminiのような「親切で物知りなAIアシスタント」ではありません。
ユーザーが科学・哲学などの小難しい質問を投げかけても、設定された人物の性格（知的好奇心、計画性、社交性、情緒など）や年齢の範囲を超えて流暢に解説を始めることはありません。
代わりに「そんな難しいこと俺に聞くなよ」「えっと、何ですかそれ…？」といった、そのキャラクターならではの**「生身の人間らしいリアルな反応（拒絶・困惑・無関心）」**を返します。

---

## 🛠️ 主な機能

### 1. 性格プロファイリング＆カスタマイズ
- **ビッグファイブ・スライダー**: 5つの因子（開放性、誠実性、外向性、協調性、神経症傾向）を0〜100%の範囲で手動調整できます。
- **性別 & 年齢設定**: 男性/女性/その他（ノンバイナリ）の切り替えと、10〜90歳までの年齢スライダーを完備。
- **レーダーチャート (Chart.js)**: 変更した性格特性がリアルタイムで美しいグラフに反映されます。
- **ランダム生成**: ボタン一つで、性格・性別・年齢がバランスよくランダム作成されます。

### 2. コンテキスト（背景環境）シミュレーション
以下のシチュエーションを選択でき、環境ごとの物理的・心理的制約がAIのセリフや話し方に強く反映されます：
- 🏫 **学校の教室**: クラスの騒がしさや、年齢相応のフランクな話し方。
- 🏢 **静かなオフィス**: 周囲に配慮した小声、タイピング音の補足、ビジネス敬語。
- ⛪ **静まり返った教会**: 静粛な礼拝堂内のため、常に「(ヒソヒソ声で)」話す制約。
- 🏭 **騒音の大きい工場**: 機械音が激しいため、常に叫び声「(大声で) なんだって！？」になる制約。
- 🏮 **賑やかな居酒屋**: お酒が入ったフレンドリーで大雑把なトーン。
- ⚙️ **カスタム設定**: 自由に入力した場所の設定がプロンプトへ引き継がれます。

### 3. APIキー消費を抑える手動トリガー設計
- パラメータの変更やランダム生成を行った段階では、**ローカルの超高速モックエンジン**が簡易解説を組み立てて即時表示（API消費はゼロ）。
- リッチなAIによる詳細な性格分析を行いたい場合のみ、カード上部の **「性格分析を開始」** ボタンを押してAPI通信を走らせる仕組みです。

### 4. 2つの対話モード
- **Gemini API モード**: APIキーを設定することで、高度なペルソナロールプレイを実行。
- **ローカルモックモード**: オフラインやAPI上限時でも動作する応答システム。設定パラメータに基づき、一人称（「俺」「あたし」「わし」「自分」）や語尾のニュアンス（10代の若者言葉 vs 65歳以上のシニア語尾）が動的に置換・生成されます。

### 5. 音声読み上げ機能 (TTS / Text-to-Speech)
- Web Speech APIを利用して、AIのセリフを音声出力します。
- 性格の「外向性」が高いと話す速度（Rate）が速くなり、「神経症傾向」が高いとおどおどした高音（Pitch）に変わります。
- 工場（大声）や教会（ヒソヒソ声）など、環境音に合わせて音量（Volume）も自動調整されます。

## 🚀 はじめ方（ローカル実行手順）

本アプリはHTML/CSS/JavaScriptのみで構成されているため、特別なビルド環境は不要です。

1. このプロジェクトフォルダに移動します。
2. ローカルサーバーを起動します（Pythonがインストールされている場合、以下のコマンドで起動できます）。
   ```bash
   python -m http.server 8000
   ```
3. ブラウザを開き、以下のURLにアクセスします。
   ```
   http://localhost:8000
   ```
